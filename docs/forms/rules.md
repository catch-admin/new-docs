# 表单规则

## alpha

只包含字母

## alpha_dash

只包含字母、数字、破折号（ - ）和下划线（ \_ ）

## alpha_num

包含字母和数字

## email

邮箱格式不正确

## integer

数据只能是整型

## ip

ip 地址格式不正确

## ipv4

ipv4 地址格式不正确

## numeric

必须是数字

## mobile

手机号格式不正确

## chinese_character

只允许中文

## strong_password

必须包含大小写字母和数字的组合，不能使用特殊字符，长度在 8-20 之间

## password

以字母开头，长度在 6~18 之间，只能包含字母、数字和下划线

## idcard

身份证格式不正确
